# Task

Below is an accurate transcription of a handwritten, typewritten or printed historical document. Your task is to enrich the transcription with basic TEI XML markup. Omit the `<teiHeader>` and just answer with the text `<body>`. A facsimile image of the original document is attached for reference.

## Transcription:

```
{{AI_TRANSCRIPTION}}
```

## Don’t encode:

- text divisions (`<div>`)

## Encode:

- headings (`<head>`)
- paragraphs (`<p>`; add `rend="noIndent"` to paragraphs that do not have first line indentation)
- page beginnings (`<pb/>`; if no page number is visible in the facsimile, omit @n)
- line beginnings (`<lb/>`)
- footnotes (`<note>`)
- line groups (`<lg>`)
- tables (`<table>`)
- lists (`<list>`)
- highlights (`<hi>`; bold, italics, etc.)

## Important:

- Note that the `<lb/>` element marks the beginning of lines and should appear at the start of lines, not at the end.
- Do not alter the transcribed text.
- Maintain the line breaks.
- Do not add text that is not in the transcription (headers, footers, and marginalia might have been omitted on purpose).
